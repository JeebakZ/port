Portfolio

Top 5 European League (2023-24) Players Scouting Application

This application helps in Scouting and analyzing players' performances across the top 5 European leagues (Premier League, La Liga, Bundesliga, Serie A, and Ligue 1) by evaluating their Passing, Crossing Skills, and Expected Goals and Assists. It aids teams in building optimal squads using machine learning. The application is built in Tableau using the 2023-24 season players' performance data from FBREF.

Foo-py: Football Analysis Web Application

Foo-py helps football analysts and coaches gain deep insights into match performance and player dynamics by visualizing passing networks, player passes, and heatmaps, using data from matches such as the Turkey vs Portugal Euro 2024 game.

Football-Computer Vision
